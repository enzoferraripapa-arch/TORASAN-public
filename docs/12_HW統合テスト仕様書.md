## 1. テスト方針

### 1.1 テスト目的

PH-07 HW設計書で定義されたHW要求（HW-REQ-001〜010）およびPH-06 システム設計書で定義されたHW/SWインタフェース（IF-01〜IF-12）に対し、ターゲットボード上でHW/SW統合テストを実施し、HW/SWの界面が仕様通りに機能することを検証する。

### 1.2 テスト戦略

- **テスト手法**: HW/SWインタフェース検証（実機計測ベース）
- **カバレッジ目標**: HW-REQ 10件全件カバー、IF 12件全件カバー
- **テスト分類**: 正常系 / 境界値 / 異常系（故障注入）
- **判定基準**: product_spec記載の定量値に基づく計測判定
- **統合戦略**: ボトムアップ（HAL→BSW→APP順にHW接続を追加）

### 1.3 テスト環境

| 項目 | 内容 |
|------|------|
| ターゲット | Renesas RL78/G14 (R5F104BG) 評価ボード |
| デバッガ | E2 Lite (Renesas) |
| オシロスコープ | 4ch デジタルオシロ（PWM波形・タイミング計測） |
| 電流発生器 | プログラマブル電流源（シャント電流注入用） |
| 電圧発生器 | プログラマブル電源（VDD可変用） |
| 信号発生器 | ファンクションジェネレータ（ホールセンサ模擬） |
| マルチメータ | 高精度DMM（DC電圧・電流計測） |
| UART端末 | シリアルモニタ（9600bps通信確認） |

## 3. テストケース詳細

### 3.1 ADC電流計測系（IF-05, HW-REQ-005, TSR-004）

#### TC-301: ADC電流 定格6A計測精度確認（正常）

| 項目 | 内容 |
|------|------|
| **テストID** | TC-301 |
| **対象IF/HW-REQ** | IF-05 / HW-REQ-005 |
| **関連TSR** | TSR-004 |
| **カテゴリ** | 正常 |
| **優先度** | 高 |

**前提条件**: RL78/G14ターゲットボード通電、SW起動済み、ADC ch0初期化済み

**入力**:
- 電流発生器でシャント抵抗に定格6A相当電流を印加
- シャント電圧: 6 × 0.01 = 0.06V → オペアンプ出力: 0.06 × 50 = 3.0V

**テスト手順**:
1. 電流発生器を6A出力に設定
2. ADC ch0 の読取値を10回連続取得
3. 平均値と設計値614の差分を算出
4. マルチメータで実際のオペアンプ出力電圧を並行計測

**期待結果**:
- ADC読取値平均 = 614 ± 5LSB（609〜619）
- マルチメータ計測値 = 3.0V ± 0.05V

**PASS/FAIL判定基準**:
- |ADC平均 - 614| ≦ 5

**テスト結果**: 実施待ち

**安全分類**: Class B
**トレーサビリティ**: IF-05 → HW-REQ-005 → TSR-004 → FSR-003 → SG-002
