# トレーサビリティマトリクス
## 洗濯機BLDCモータ制御ECU

文書番号: WMC-TRACE-001 / 版数: 2.1 / 作成日: 2026-03-02

---

## 1. 概要

本文書は、ISO 26262 Part 8 §6 に基づき、安全要件のトレーサビリティを管理する。PH-03（HARA）時点では HZ → SG の上位トレースを確立し、以降のフェーズで FSR → TSR → SR → TC の下位展開を追記する。

### ID体系

| レベル | ID形式 | 定義フェーズ | 説明 |
|--------|--------|-----------|------|
| ハザード | HZ-xx | PH-02/03 | 同定されたハザード |
| 安全目標 | SG-xxx | PH-03 | ハザードから導出された安全目標 |
| 機能安全要求 | FSR-xxx | PH-04 | 安全目標を実現する機能要求 |
| 技術安全要求 | TSR-xxx | PH-05 | 機能安全要求の技術的実現 |
| SW安全要求 | SR-xxx | PH-08 | ソフトウェアレベルの安全要求 |
| テストケース | TC-xxx | PH-11 | 要求を検証するテスト |
| セキュリティ目標 | SecG-xxx | PH-03/04 | 脅威から導出されたセキュリティ目標 |
| セキュリティ要求 | SecR-xxx | PH-04/05 | セキュリティ目標の実現要求 |
