# システム要件仕様書（System Requirements Specification）
## 洗濯機BLDCモータ制御ECU

文書番号: WMC-SRS-SYS-001 / 版数: 1.0 / 作成日: 2026-03-02

**適用規格**: IEC 60730-1:2023 Class B (製品安全分類) + IEC 60335-2-7 (製品安全)
**プロセス基盤**: ISO 26262:2018 の手法を活用 / **SPICE**: SYS.2 WP 17-11
**対象MCU**: Renesas RL78/G14 (R5F104BG) — Flash 64KB / RAM 5.5KB / 32MHz

## 1. 目的と適用範囲

本文書は Automotive SPICE SYS.2 プロセスのワークプロダクト WP 17-11「システム要件仕様」として、洗濯機BLDCモータ制御ECUの全システム要件を一覧化する。

要件の導出根拠と安全コンセプトの詳細は以下の文書を参照:
- `docs/04_FSC.md` — 機能安全コンセプト（FSR の導出根拠・FTTI分析・安全アーキテクチャ）
- `docs/05_TSC.md` — 技術安全コンセプト（TSR の詳細技術仕様・DC見積り・HW/SW割当）

| FSR-003 | 過電流検出機能 | 電流センサADCで10ms周期監視、8A超過を検出 | Class B | SG-002 | 検出 | SW+HW | テスト |
| FSR-004 | 過電流時通電遮断機能 | 過電流検出時≦50msで全相ゲートOFF | Class B | SG-002 | 反応 | SW | テスト |
| FSR-005 | 蓋開放検出機能 | 蓋センサGPIOを5ms周期で監視、5ms以上継続開を検出 | Class B | SG-003 | 検出 | SW | テスト |
| FSR-006 | 蓋開放時モータ停止機能 | 蓋開放検出時≦200msでモータ停止 | Class B | SG-003 | 反応 | SW | テスト |

| FSR-009 | 電圧監視機能 | MCU VDD電圧を100ms周期で監視、4.5V〜5.5V範囲外を検出 | Class B | SG-006 | 検出 | SW+HW | テスト |

| TSR-003 | PWM出力即時停止制御 | 全PWMチャネル強制クリア+ゲートEN=LOW、≦100ms | Class B | FSR-002 | SW+HW | 高 | テスト |
| TSR-004 | ADC電流値取得・過電流判定 | ADC 10bit/5V、100Hz、移動平均N=4、8A閾値判定 | Class B | FSR-003 | SW | 高 | テスト |
| TSR-005 | インバータゲート遮断制御 | 全相OFF+HWリセット信号(P10)、≦50ms | Class B | FSR-004 | SW+HW | 高 | テスト |

| TSR-015 | WDTタイムアウト設定・キック管理 | RL78/G14内蔵WDT、100msタイムアウト、50msキック | Class B | FSR-007/008 | SW+HW | 高 | テスト |
