+16.1 マニュアル再生成
+フレームワークの構成（スキル追加・プロセス変更等）に変更があった場合、Word 文書を再生成して最新状態に同期する。
+マニュアル再生成手順
+Step 1: コンテンツ生成スクリプト（generate_spec_doc.py / generate_ops_manual.py）を編集する
+Step 2: 統合スクリプトを実行する
+Step 3: 差分レポートを確認する
+Step 4: 変更をコミットする
+$ python scripts/generate_manuals.py

+16.2 差分レポートの確認
+再生成時に差分が検出されると、docs/manuals/diff_reports/ に unified diff 形式のレポートが保存される。
+レポートの読み方:
++ で始まる行: 追加された内容
+- で始まる行: 削除された内容
+@@ で始まる行: 変更セクションの位置情報
+16.3 バージョン履歴の確認
+.manuals_manifest.json にバージョン番号・ハッシュ・差分統計が蓄積される。過去の変更履歴を遡る場合はこのファイルを参照する。
+$ python scripts/generate_manuals.py --diff-only
