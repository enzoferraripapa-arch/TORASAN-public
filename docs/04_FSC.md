### 1.2 SG-004（漏電）のHW対策記録

SG-004（漏電検出時即座に電源遮断）は、以下の理由によりSW FSRを設定せず、HW対策のみとする:

- 漏電はモータ巻線・ケーブルの絶縁劣化に起因し、SWでの直接検出は不可能
- 漏電遮断器（RCD: 30mA/30ms）による即時遮断がISO/IEC要件を満たす
- SW範囲の対応: 絶縁異常の間接的な兆候（電圧異常等）はSG-006の範囲で監視

**外部リスク低減手段**: RCD（漏電遮断器）— 動作電流≦30mA、応答時間≦30ms

## 2. 機能安全要求（FSR）詳細

### FSR-001: 過速度検出機能

| 項目 | 内容 |
|------|------|
| 親SG | SG-001 |
| 安全分類 | IEC 60730 Class B |
| 種別 | 検出 |
| 実現形式 | SW |

**機能説明**: モータ回転速度を周期的に監視し、過速度閾値の超過を検出する。

**定量仕様**:
- 監視対象: ホールセンサ（IF-02〜04）からの回転速度
- 過速度閾値: **1500rpm** (`product_spec.motor.overspeed_rpm`)
- 監視周期: **10ms** (ADC電流サンプリング100Hzと同期)
- 検出遅延: ≦1監視周期（10ms）

**トリガ条件**: OP-01/02/03（モータ運転中）で常時実行

---

### FSR-002: 過速度時モータ停止機能

| 項目 | 内容 |
|------|------|
| 親SG | SG-001 |
| 安全分類 | IEC 60730 Class B |
| 種別 | 反応 |
| 実現形式 | SW |

**機能説明**: FSR-001が過速度を検出した場合、規定時間内にインバータを停止しモータを安全状態にする。

**定量仕様**:
- 応答時間: **≦100ms**（検出から停止完了まで）— `product_spec.wdt.timeout_ms`
- 安全動作: PWM出力 = 0%、インバータゲート全OFF
- 状態保持: OP-05（異常停止）に遷移、電源リセットまで保持
- UI通知: エラーコード送信（IF-09 UART）

---

### FSR-003: 過電流検出機能

| 項目 | 内容 |
|------|------|
| 親SG | SG-002 |
| 安全分類 | IEC 60730 Class B |
| 種別 | 検出 |
| 実現形式 | SW+HW |

**機能説明**: モータ駆動電流を監視し、過電流閾値の超過を検出する。

**定量仕様**:
- 監視対象: 電流センサADC（IF-05, ANI0）
- 過電流閾値: **8A (8000mA)** (`product_spec.motor.max_a` / `max_ma`)
- 監視周期: **10ms** (`product_spec.adc.current_hz` = 100Hz)
- ADC分解能: 10bit / Vref 5.0V → 約4.88mV/LSB
- HWバックアップ: ヒューズによる物理遮断（独立）

---

### FSR-004: 過電流時通電遮断機能

| 項目 | 内容 |
|------|------|
| 親SG | SG-002 |
| 安全分類 | IEC 60730 Class B |
| 種別 | 反応 |
| 実現形式 | SW |

**機能説明**: FSR-003が過電流を検出した場合、規定時間内に通電を遮断する。

**定量仕様**:
- 応答時間: **≦50ms**（検出から遮断完了まで）
- 安全動作: PWM出力 = 0%、全相ゲートOFF
- 状態保持: OP-05に遷移
- ログ記録: 過電流イベント（時刻、電流値）

---

### FSR-005: 蓋開放検出機能

| 項目 | 内容 |
|------|------|
| 親SG | SG-003 |
| 安全分類 | IEC 60730 Class B |
| 種別 | 検出 |
| 実現形式 | SW |

**機能説明**: 蓋センサの状態を監視し、蓋の開放を検出する。

**定量仕様**:
- 監視対象: 蓋センサGPIO（IF-07, P43, pull-up）
- 信号: 蓋閉=LOW / 蓋開=HIGH
- チャタリング対策: 5ms以上継続して開状態を確認後に判定
- 監視周期: **5ms**（制御周期以下）
- 蓋センサ診断: 定期的な動作確認（診断周期≦制御周期）

---

### FSR-006: 蓋開放時モータ停止機能

| 項目 | 内容 |
|------|------|
| 親SG | SG-003 |
| 安全分類 | IEC 60730 Class B |
| 種別 | 反応 |
| 実現形式 | SW |

**機能説明**: FSR-005が蓋開放を検出した場合、規定時間内にモータを停止する。

**定量仕様**:
- 応答時間: **≦200ms**（検出からモータ停止まで）
- 安全動作: PWM出力 = 0%
- 状態保持: 蓋が閉じられるまで停止継続（OP-05）
- 蓋ロック: IF-08（P44）でロック状態維持
- UI通知: 蓋開放表示

---

### FSR-007: MCU自己診断機能

| 項目 | 内容 |
|------|------|
| 親SG | SG-005 |
| 安全分類 | IEC 60730 Class B |
| 種別 | 検出 |
| 実現形式 | SW |

**機能説明**: IEC 60730 Annex H に準拠し、MCU内部の異常を周期的に検出する。

**診断項目**:

| テスト | スタートアップ | ランタイム | 手法 | パラメータ |
|--------|-----------|---------|------|----------|
| CPU | 命令実行確認 | レジスタテスト | IEC 60730 H.2.16.5 | — |
| RAM | 全域初期化 | March C（分割） | IEC 60730 H.2.19.6 | 256B × 22ブロック / 全域2.2秒 |
| ROM | CRC32全域 | CRC32分割検証 | IEC 60730 H.2.19.8 | `product_spec.crc_algorithm` |
| クロック | 周波数確認 | 独立クロック比較 | IEC 60730 H.2.18.10 | 許容: ±4% (`product_spec.clock_tolerance_pct`) |
| ADC | リファレンス | プラウジビリティ | IEC 60730 H.2.18.1 | 100Hz/10Hz |

**ランタイム診断周期**: 10ms周期で段階的実行（全テスト完了: RAM 2.2秒 / ROM: Flash 64KB依存）

---

### FSR-008: MCU異常時安全遷移機能

| 項目 | 内容 |
|------|------|
| 親SG | SG-005 |
| 安全分類 | IEC 60730 Class B |
| 種別 | 反応 |
| 実現形式 | SW |

**機能説明**: FSR-007が異常を検出した場合、またはWDTタイムアウトが発生した場合、安全状態へ遷移する。

**定量仕様**:
- 応答時間: **≦100ms** (`product_spec.wdt.timeout_ms`)
- WDTキック周期: **50ms** (`product_spec.wdt.kick_ms`)
- 安全動作: PWM全相OFF、全出力無効化
- 状態保持: 異常フラグ設定、電源リセットまで保持
- 復帰: 電源OFF→ON（ソフトリセット不可）→ スタートアップ診断PASS → OP-04
- UI通知: エラーコード＋赤色LED点滅

**トリガ条件**:
- FSR-007 自己診断の異常検出
- WDTタイムアウト（100ms無キック）
- スタックオーバーフロー検出
- 電源断後の不正再起動（HZ-05サブケース）

---

### FSR-009: 電圧監視機能

| 項目 | 内容 |
|------|------|
| 親SG | SG-006 |
| 安全分類 | IEC 60730 Class B |
| 種別 | 検出 |
| 実現形式 | SW+HW |

**機能説明**: MCU電源電圧を監視し、許容範囲外を検出する。

**定量仕様**:
- 監視対象: MCU VDD（IF-06, ANI1）
- 正常範囲: **4.5V〜5.5V** (`product_spec.voltage_monitor.min_v` / `max_v`)
- 監視周期: **100ms** (10Hz = `product_spec.adc.voltage_hz`)
- ADC分解能: 10bit / Vref 5.0V
- HWバックアップ: 電圧監視IC（リセット生成）

---

### FSR-010: 電圧異常時モータ禁止機能

| 項目 | 内容 |
|------|------|
| 親SG | SG-006 |
| 安全分類 | IEC 60730 Class B |
| 種別 | 反応 |
| 実現形式 | SW |

**機能説明**: FSR-009が電圧異常を検出した場合、モータ動作を禁止する。

**定量仕様**:
- 安全動作: PWM出力コマンドを全て遮断
- 状態保持: 電圧が正常範囲に復帰するまで禁止継続
- ブラウンアウト対策: 電圧低下時はモータ停止を優先（MCU動作維持より安全）
- UI通知: 電圧異常エラー表示
- ログ: 電圧異常イベント（時刻、電圧値）

## 3. フォールトトレラント時間間隔（FTTI）分析

| SG | 危険事象 | FTTI | 診断周期 | 検出遅延 | 反応時間 | 余裕 | 判定 |
|----|---------|------|---------|---------|---------|------|------|
| SG-001 | 過速度→ドラム破損 | 500ms | 10ms | ≦10ms | ≦100ms | 380ms | **OK** |
| SG-002 | 過電流→発火 | 200ms | 10ms | ≦10ms | ≦50ms | 130ms | **OK** |
| SG-003 | 蓋開放→接触 | 500ms | 5ms | ≦10ms | ≦200ms | 285ms | **OK** |
| SG-005 | MCU暴走→制御喪失 | 200ms | 50ms(WDT) | ≦50ms | ≦100ms | 0ms | **境界** |
| SG-006 | 電圧異常→制御喪失 | 500ms | 100ms | ≦100ms | 即時 | 300ms | **OK** |

**注**: SG-005 は FTTI ≒ 診断周期 + 反応時間で余裕が最小。WDTタイムアウト100msが最終防壁として機能。

## 4. 安全アーキテクチャ概念

### 4.1 フェールセーフ原則

**安全状態**: モータ停止（PWM全相OFF、インバータゲート無効化）= OP-05

**原則**:
- 異常検出時は即座にモータ停止
- 一度安全状態に遷移したら電源リセットまで保持
- UI通知でユーザーに異常を認知させる

### 4.2 多層防御構造

```
  層1: SW安全監視                層2: HW安全機構
  ┌──────────────────┐          ┌──────────────────┐
  │ FSR-001 過速度検出 │          │ 機械的バランサー   │
  │ FSR-003 過電流検出 │          │ ヒューズ（過電流）  │
  │ FSR-005 蓋開放検出 │          │ 電圧監視IC         │
  │ FSR-007 MCU診断    │          │ WDT（HW）          │
  │ FSR-009 電圧監視   │          │ RCD（漏電遮断器）   │
  │         ↓          │          │ 蓋ロック機構       │
  │ FSR-002/004/006/  │          │         ↓          │
  │ 008/010 反応機能   │          │ 物理的安全遮断     │
  └──────────────────┘          └──────────────────┘
           ↓                             ↓
    OP-05（異常停止）              独立した安全確保
```

### 4.3 冗長性設計

| 安全機能 | 層1（SW） | 層2（HW） | 共通原因対策 |
|---------|----------|----------|------------|
| 過速度 | FSR-001/002 | 機械的バランサー | SW/HW独立設計 |
| 過電流 | FSR-003/004 | ヒューズ | 異なる検出原理 |
| 蓋開放 | FSR-005/006 | 蓋ロック機構 | 機械＋電子 |
| MCU異常 | FSR-007/008 | WDT | 内部診断＋外部監視 |
| 電圧異常 | FSR-009/010 | 電圧監視IC | ADC＋専用IC |
| 漏電 | — | RCD | HW専用 |

## 6. 外部リスク低減手段の前提

本FSCは以下のHW/機械機構が確実に機能していることを前提とする。

| # | 手段 | 仕様 | 対応SG | 役割 |
|---|------|------|--------|------|
| 1 | 漏電遮断器（RCD） | 動作電流≦30mA / 応答≦30ms | SG-004 | 感電防止（HW主体） |
| 2 | ヒューズ | モータ電流定格に合わせた溶断値 | SG-002 | 過電流HWバックアップ |
