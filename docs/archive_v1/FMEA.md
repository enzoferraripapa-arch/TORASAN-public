# FMEA（故障モード影響解析）— 軽量版

**適用規格**: IEC 60730 クラスB / **プロセッサ**: RL78/RX
**文書番号**: WMC-FMEA-001 / **版数**: 1.0 / **作成日**: 2026-02-27
**ステータス**: 暫定（EXPLORE モード軽量版）

---

## 概要

本FMEA(Failure Mode and Effects Analysis)は、洗濯機モータ制御ユニット（脱水機能含む）の故障モード及び影響を分析するもの。
IEC 60730 クラスBの要求に基づき、重大度が高い故障及び検出困難な故障に焦点を当てたものである。

## 凡例

- **DC(Detection Confidence)**: 故障が検出される可能性の程度
  - 高(90%以上): 検出される可能性が高い
  - 中(60%程度): 検出されるが遅延の可能性
  - 低(～50%): 検出困難

- **深刻度**: IEC 60730に基づく分類
  - **高**: 重大事故、人身事故、装置破損の可能性
  - **中**: 機能喪失、軽微な破損
  - **低**: 可用性低下のみ

- **安全対策**: SR(Safety Requirement)の参照と対応する設計・実装対策
