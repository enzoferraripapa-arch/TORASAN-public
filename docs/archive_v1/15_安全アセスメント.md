## 1. 安全ケース概要・エグゼクティブサマリー

### 1.1 プロジェクト概要

| 項目 | 内容 |
|------|------|
| **製品名** | 洗濯機モータ制御ユニット |
| **用途** | 家庭用洗濯機の主駆動モータ制御 |
| **定格** | 200V / 1.5kW 三相 or 単相 |
| **マイコン** | Renesas RL78/RX シリーズ |
| **適用規格** | IEC 60730-1/-2-13, IEC 60335-2-7 |
| **安全完全性レベル** | Class B (IEC 60730基準) |
| **開発プロセス** | ISO 26262 準拠のプロセス（簡略化） |

### 1.2 安全ケース構成

本システムは以下の階層構造で安全性を確保する：

```
安全目標 (SG)
  ↓
機能安全要求 (FSR)
  ↓
技術安全要求 (TSR)
  ↓
ソフトウェア安全要求 (SR) + 設計要求 (DR)
  ↓
テストケース (TC)
```

**構成元数**:
- 安全目標 (SG): **6件**
- 機能安全要求 (FSR): **10件**
- 技術安全要求 (TSR): **15件**
- ソフトウェア安全要求 (SR): **16件**
- 設計要求 (DR): **4件**
- テストケース (TC): **48件** (HW統合:18件 + システム:30件)

### 1.3 安全ケーススコープ

**対象システム範囲**：
- インバータ回路 (MOSFET + Gate Driver)
- 電流/電圧/温度センサ回路
- MCU (RL78) ファームウェア
- ユーザインターフェース (LCD/ボタン)
- 蓋ロック機構
- WDT + リセット回路

**対象外**：
- 遠隔通信機能 (将来拡張)
- クラウド連携機能 (将来拡張)
- 機械部品の強度設計（別文書）

## 13. 補足・注記

### 13.1 用語解説

| 用語 | 説明 |
|------|------|
| **SG** | 安全目標 (Safety Goal) - システムが満たすべき安全要求の最上位レベル |
| **FSR** | 機能安全要求 (Functional Safety Requirement) - SG を実装する機能レベルの要求 |
| **TSR** | 技術安全要求 (Technical Safety Requirement) - FSR をハードウェア/ソフトウェアで実装する技術的要求 |
| **SR** | ソフトウェア安全要求 (Software Requirement) - MCUファームウェアに対する要求 |
| **DR** | 設計要求 (Design Requirement) - 汎用設計方針 |
| **TC** | テストケース (Test Case) - 検証・妥当性確認の試験項目 |
| **Class B** | IEC 60730 の安全完全性レベル分類。簡易的な自己診断により一定の保護を実現 |
| **FMEA** | Failure Mode & Effect Analysis - 故障モード影響分析 |
| **FTA** | Fault Tree Analysis - フォルトツリー分析 |
| **TCL** | Tool Capability Level - 設計支援ツールの適格性レベル (TCL1～TCL3) |
| **暫定** | 本文書は探索/開発段階の暫定版。最終的な承認は Phase 7 で実施 |
