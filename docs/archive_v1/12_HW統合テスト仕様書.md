# HW統合テスト仕様書

**文書ID**: WMC-HIT-001
**バージョン**: v1.0
**作成日**: 2026-02-27
**適用規格**: IEC 60730 クラスB / 暫定
**プロジェクト**: 洗濯機モータ制御システム (RL78/RX)

---

## 1. テスト方針・環境

### 1.1 テスト目的
HW+SW統合環境においてシステムレベルの安全機能を検証する。実機基板と実モータ（又は模擬負荷）を用いて、各HW安全機構の動作確認及び精度検証を行う。

### 1.2 テスト環境構成
- **テスト対象**: 洗濯機モータ制御基板（RL78マイコン搭載）
- **電源**: 直流電源 200V±10% または商用三相電源
- **負荷**: 実モータ（定格出力 1.5kW以上）または トルク計付き模擬負荷
- **計測装置**: デジタルオシロスコープ、電流プローブ、電圧計、タコメータ
- **制御/監視**: UART デバッグポート、タッチパネル UI

### 1.3 安全適合性確認の範囲
- IEC 60730-1 (汎用要件) 準拠状況
- IEC 60730-2-13 (マイコン機器) 準拠状況
- IEC 60335-2-7 (洗濯機安全規格) との整合性
- Class B 機能安全レベルの達成確認

## 4. テスト実施上の注意事項

### 4.1 安全上の注意
- 高圧・高電流環境のため、作業者は電気技師資格を要する
- 全テスト実施時は感電防止ガード、アースを施す
- 非常停止ボタンを常時スタンバイ状態に保つ

### 4.2 測定機器の管理
- テスタ、オシロスコープは年1回以上の校正を実施
- 計測精度の記録を保持し、テスト合否判定の証跡とする

### 4.3 再現性確保
- テスト条件（温度、湿度、電源安定度）を記録
- 異常発生時は、条件を完全に再現可能にする
