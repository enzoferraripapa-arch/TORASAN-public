# SWユニットテスト仕様書
## 洗濯機モータ制御 ユニットテスト仕様

**適用規格**: IEC 60730 クラスB
**対象システム**: RL78/RX ベース洗濯機モータ制御
**文書番号**: WMC-UT-001
**版数**: 1.0
**作成日**: 2026-02-27
**ステータス**: 暫定版

---

## 1. テスト方針

### 1.1 テスト種別
本仕様書は、洗濯機モータ制御ソフトウェアの**ホワイトボックステスト**に関する要件を定義する。
構造カバレッジを重視し、単体（ユニット）レベルでの徹底的なテストを実施する。

### 1.2 IEC 60730 クラスB 適用

| 項目 | 要件 | 本プロジェクト適用 |
|-----|------|------------------|
| ステートメントカバレッジ (C0) | 推奨 | **100%** |
| ブランチカバレッジ (C1) | 推奨 | **90%以上** |
| MC/DC | 不要 | 対象外（クラスC以上） |
| 境界値テスト | 必須 | 全ての値判定ロジック対象 |
| 異常系テスト | 必須 | 自己診断・フェイルセーフ機能確認 |

### 1.3 テスト環境

- **テストフレームワーク**: Unity / CMock
- **実行環境**: ホストPC上でのクロスコンパイル実行
- **ビルド環境**: GCC クロスコンパイラ (arm-none-eabi)
- **カバレッジ計測**: gcov, lcov
- **テスト自動化**: CIパイプライン統合

### 1.4 スタブ/モック戦略

HAL層（ハードウェア抽象化層）は完全にモック化し、以下に焦点を当てる：

- モータ制御ロジック（RPM計算、速度監視）
- 電流・電圧監視の安全判定ロジック
- 蓋センサの状態遷移管理
- 自己診断機能（RAM/ROM/クロック検査）
- 安全状態遷移とフェイルセーフ動作

## 3. カバレッジ目標

### 3.1 構造カバレッジ要件

| カバレッジ種別 | 目標値 | IEC 60730 クラスB 要求 | 測定方法 |
|---------------|--------|----------------------|----------|

| **C1（ブランチ）** | **90%以上** | 推奨 | gcov ブランチカバレッジ |
| **MC/DC（修正条件判定）** | 対象外 | クラスC以上で要求 | N/A |

### 3.2 カバレッジ測定フロー

```
1. テストビルド: -fprofile-arcs -ftest-coverage フラグで GCC コンパイル
2. テスト実行: Unity テストフレームワークで全 TC 実行
3. カバレッジ計測: gcov でカバレッジデータ生成
4. レポート生成: lcov で HTML レポート作成
5. 達成確認: C0≥100%, C1≥90% を確認
6. 未カバー行調査: 不可到達コード/例外処理を文書化
```

### 3.3 カバレッジ対象ファイル

| ファイル | 機能 | C0 目標 | C1 目標 |

| `motor_control.c` | モータ制御・RPM計算 | 100% | 90% |

| `current_monitor.c` | 電流監視・過電流検出 | 100% | 90% |

| `lid_sensor.c` | 蓋センサ入力・デバウンス | 100% | 90% |

### 3.4 不可到達コード（未カバー許可）

- ハードウェア割り込み内でのみ実行される処理（ユニットテストでは割り込み未発火）
- リセット後の初期化ルーチンの一部（テスト環境では再現不可）

/* test_motor_control.c */
#include "unity.h"
#include "motor_control.h"
#include "mock_hal_timer.h"
#include "mock_hal_pwm.h"

void setUp(void) {
    // 初期化
    mock_hal_timer_Init();
    mock_hal_pwm_Init();
}

void tearDown(void) {
    // クリーンアップ
}

/* TC-001: ホールセンサ 1ms 周期入力で RPM=60000 算出 */
void test_motor_rpm_calculation_60000rpm(void) {
    // Arrange: モック設定
    mock_hal_timer_SetPulsePeriod_ms(1);

    // Act: テスト対象関数呼び出し
    uint16_t rpm = GetMotorRPM();

    // Assert: 期待値確認
    TEST_ASSERT_UINT16_WITHIN(600, rpm, 60000);  // ±1% = ±600rpm
}

/* TC-002: ホールセンサ未入力で RPM=0 */
void test_motor_rpm_stopped(void) {
    // Arrange
    mock_hal_timer_SetTimeout_ms(50);

    // Act
    uint16_t rpm = GetMotorRPM();

    // Assert
    TEST_ASSERT_EQUAL_UINT16(0, rpm);
}
