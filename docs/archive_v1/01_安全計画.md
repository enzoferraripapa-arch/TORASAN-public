# 安全計画書
## 洗濯機モータ制御システム開発

---

## 1. プロジェクト概要と安全計画の適用範囲

### 1.1 プロジェクト概要
本プロジェクトは、家庭用電気器具である洗濯機のモータ制御システムの開発を対象としています。

**プロジェクト名称：** 洗濯機モータ制御システム（WMC: Washing Machine Control）
**対象製品：** 家庭用洗濯機
**制御対象：** モータ駆動制御、センサ入力処理、安全機能実装
**プロセッサ：** Renesas RL78/G14 (R5F104BG)

### 1.2 安全計画の適用範囲
本安全計画は、洗濯機モータ制御システムの開発における以下の活動に適用されます：

- ソフトウェア設計・実装
- 安全機能の仕様策定と検証
- 自動診断機能の実装と妥当性確認
- 全開発フェーズにおける安全活動

本計画は、IEC 60730 クラスB に準拠した開発プロセスに基づいています。

## 3. 適用規格・安全レベル目標

### 3.1 適用規格

**IEC 60730-1:2023** - 家庭用および類似一般用電気器具の自動電気制御
- **適用クラス：クラスB**（安全自動制御機能付き）
- **特性：** 完全な自己診断機能を備えた安全制御システム

**IEC 60730 附属書H** - ソフトウェア要件
- ソフトウェアセーフティレベルの定義
- 自動診断機能の実装要件
- 信頼度レベル（TCL）の適用

### 3.2 安全レベル目標

| 項目 | 規格 | 目標レベル |
|-----|------|---------|
| 機能安全クラス | IEC 60730-1 | クラスB |
| 自動診断機能 | IEC 60730-1 | 必須実装 |
| ソフトウェア信頼度レベル（TCL） | IEC 60730-1, Annex H | TCL3（最高レベル） |
| コード検査方法 | IEC 60730-1 Annex H | 形式手法またはレビュー |

### 3.3 自動診断機能（Annex H）

IEC 60730 Annex H に準拠した以下の自動診断機能を実装：

- **入力診断：** センサ回路の短絡・開放検出
- **出力診断：** モータドライバ出力の異常検出
- **メモリ診断：** ROM/RAM の破損検出（チェックサム）
- **プロセッサ診断：** CPU レジスタ異常検出
- **診断周期：** 毎制御周期または定期実施

## 6. レビュー計画

### 6.1 EXPLORE モード（開発初期段階）

本プロジェクトは EXPLORE モード で開始されます。この段階では、以下のレビュー方針が適用されます：

**EXPLORE モード レビュー：**
- Claude Code の全出力物は、開発開始前に **TORA による自己レビュー** を実施
- 重点レビュー項目：
  - IEC 60730 Annex H への適合性
  - 自動診断機能の仕様妥当性
  - コード品質（MISRA C に準拠）
  - テストケース充足性

### 6.2 CERTIFY モード（移行時）

プロジェクトが成熟段階に入る場合、**CERTIFY モード** への移行を検討します：

- **形式的なデザインレビュー** を実施
- **独立検証** を導入
- **外部専門家によるレビュー** を実施
- 完全な設計書・テスト報告書の整備

### 6.3 レビュー成果物

- 技術仕様書（SRS, SDS）
- 実装コード（ソースコード + オブジェクトコード）
- テスト計画・テスト結果報告書
- 自動診断機能検証書
- 設計変更ログ

## 7. 変更管理方針

### 7.1 設計変更プロセス

全ての安全機能に関連する変更は、以下のプロセスで管理します：

1. **変更要求の記録** → project.json の changeLog に記載
2. **影響分析** → 関連仕様・テストへの影響を確認
3. **承認** → TORA による承認取得
4. **実装・検証** → 該当コンポーネント再テスト実施
5. **ドキュメント更新** → 関連ドキュメント版数更新

### 7.2 バージョン管理

- **ドキュメント版数：** メジャー.マイナー形式（例：1.0 → 1.1 → 2.0）
- **ソースコード版数：** Git タグ版数に統一（例：v1.0.0）
- **オブジェクトコード：** ビルド番号で追跡

### 7.3 変更ログ管理

project.json に以下の形式で記録：

```json
{
  "changeLog": [
    {
      "date": "2026-02-27",
      "version": "1.0",
      "change": "初版作成",
      "approver": "TORA"
    }
  ]
}
```
