# システムテスト仕様書

**文書ID**: WMC-ST-001
**バージョン**: v1.0
**作成日**: 2026-02-27
**適用規格**: IEC 60730 クラスB / 暫定
**プロジェクト**: 洗濯機モータ制御システム (RL78/RX)

---

## 1. テスト方針

### 1.1 テスト目標
実機環境にて以下を検証する：
- 全安全目標(SG)の達成状況確認
- 機能安全要求(FSR)及び技術安全要求(TSR)の実装妥当性
- IEC 60730-1/-2-13 (一般電気機器安全規格) への準拠
- IEC 60335-2-7 (洗濯機機器の個別安全規格) への準拠
- 継続的な長時間運転での信頼性確認

### 1.2 テスト対象システム
- **構成**: MCU (RL78) + インバータHW + センサ群 + UI
- **運用環境**: 200V / 1.5kW モータ駆動
- **想定ユーザ**: 一般家庭 (Class B環境)

### 1.3 テスト環境・機器
- 実機洗濯機（又は試験台）
- 実モータ + 定格サイクル負荷シミュレータ
- 電源（200V 変動対応）
- デジタルオシロスコープ、電流計、タコメータ
- タッチパネルUI、遠隔監視システム
