## 1. アイテムの機能説明

### 1.1 概要

本アイテムは、家庭用洗濯機のブラシレスDCモータを制御するECU（Electronic Control Unit）である。洗い・すすぎ・脱水の各工程において、モータの回転数、回転方向、出力トルクを安全に制御する。

| 項目 | 仕様 |
|------|------|
| MCU | Renesas RL78/G14 (R5F104BG) — Flash 64KB / RAM 5.5KB (5,632B) / 32MHz |
| モータ | BLDC 1500W / 定格1200rpm / 過速度1500rpm / 定格6A / 最大8A |
| ADC | 10bit / Vref 5.0V / 電流100Hz / 電圧10Hz |
| WDT | タイムアウト100ms / キック50ms |
| 電圧監視 | 4.5V〜5.5V |
| 安全分類 | IEC 60730 Class B（プロセス基盤: ISO 26262 手法活用） |

### 1.2 主要機能

- **モータ回転数制御**: PWM信号による速度制御（0〜100%）
- **回転方向制御**: フォワード・リバース切替
- **トルク制御**: 電流値監視に基づく出力制限
- **センサ入力処理**: ホールセンサ(3相)、電流センサ、水位センサ、蓋ロックセンサの監視
- **UI通信**: パネルとの双方向通信（UART 9600bps）

### 1.3 安全機能（IEC 60730 Annex H 準拠）

| ID | 安全機能 | 検出手法 | 応答 |
|----|---------|---------|------|
| SF-01 | モータ過速度検出・遮断 | ホールセンサ回転数監視（閾値1500rpm） | モータ即時停止 |
| SF-02 | 過電流検出・遮断 | 電流センサADC監視（閾値8A） | モータ即時停止 |
| SF-03 | 蓋開放検出・停止 | 蓋センサ Hi/Lo 監視 | モータ停止 + 蓋ロック |
| SF-04 | WDTプログラムフロー監視 | ウォッチドッグ（100ms TO / 50ms キック） | システムリセット |
| SF-05 | 電圧異常検出 | ADC電圧監視（4.5V〜5.5V範囲外） | 安全状態遷移 |
| SF-06 | RAM破損検出 | March C（256Bブロック × 22、全域2.2秒） | 安全状態遷移 |
| SF-07 | Flash改竄検出 | CRC32 | 起動禁止 |
| SF-08 | CPU異常検出 | レジスタテスト | システムリセット |
| SF-09 | ADC異常検出 | プラウジビリティチェック（電流100Hz/電圧10Hz） | 安全状態遷移 |

## 3. 運用シナリオ定義

### 3.1 運用モード一覧

| OP-ID | モード名 | 回転数 | 回転方向 | 電流上限 | 蓋ロック | 説明 |
|-------|---------|-------|---------|---------|---------|------|
| OP-01 | 洗い | 〜400rpm | 正逆交互 | 6A | ロック | 衣類の洗浄。正逆回転を周期的に繰り返す |
| OP-02 | すすぎ | 〜400rpm | 正逆交互 | 6A | ロック | 洗剤を洗い流す。給水と排水を伴う |
| OP-03 | 脱水 | 〜1200rpm | 一方向 | 6A | ロック | 高速回転による脱水。過速度監視が最重要 |
| OP-04 | 待機 | 0rpm | — | 0A | 解除 | 電源ON・モータ停止。UIからの指令待ち |
| OP-05 | 異常停止 | 0rpm | — | 0A | ロック維持 | 安全機能発動時。モータ即時停止、蓋ロック維持、UIにエラー通知 |

### 3.2 運用シナリオ詳細

**OP-01: 洗い**
- 入力: UIパネルからの洗いコース選択
- 動作: 正回転(3秒)→停止(1秒)→逆回転(3秒)→停止(1秒) を繰り返し
- 監視: 電流≦6A、回転数≦400rpm、蓋ロック確認
- 遷移: タイマー完了→OP-02 / 異常検出→OP-05

**OP-02: すすぎ**
- 入力: OP-01完了またはUIパネルからのすすぎ選択
- 動作: 洗いと同様の正逆回転（給排水バルブ制御は本ECU範囲外）
- 監視: 電流≦6A、回転数≦400rpm、蓋ロック確認
- 遷移: タイマー完了→OP-03 / 異常検出→OP-05

**OP-03: 脱水**
- 入力: OP-02完了またはUIパネルからの脱水選択
- 動作: 一方向回転で段階的に加速（0→400→800→1200rpm）
- 監視: **過速度≦1500rpm（最重要）**、電流≦6A、蓋ロック確認、振動異常
- 遷移: タイマー完了→OP-04 / 異常検出→OP-05

**OP-04: 待機**
- 入力: 電源ON、全工程完了、またはユーザー停止操作
- 動作: モータ停止、蓋ロック解除、UI表示更新
- 監視: 電圧監視、WDT、蓋状態
- 遷移: UIコマンド受信→OP-01/02/03 / 電源OFF

**OP-05: 異常停止**
- トリガー: SF-01〜SF-09 のいずれかが発動
- 動作: モータ即時停止（PWM=0%、ブレーキ）、蓋ロック維持、UIにエラーコード送信
- 復帰: ユーザーが電源OFF→ON（リセット）後、自己診断PASSで OP-04 へ

### 3.3 モード遷移図

```
        UIコマンド        UIコマンド        UIコマンド
  [OP-04]───────→[OP-01]───────→[OP-02]───────→[OP-03]
  待機    ←───────  洗い   ←───────  すすぎ  ←───────  脱水
          完了/停止        完了/停止        完了/停止    │
    ↑                                                   │完了
    │                                                   │
    └───────────────────────────────────────────────────┘
    ↑
    │ 電源リセット後
  [OP-05] ←──── 全モードから異常検出時に遷移
  異常停止
```

## 5. 既知のハザードと安全関連法規

### 5.1 既知のハザード

| HZ-ID | ハザード種別 | 説明 | 予想される害 |
|-------|-----------|------|-----------|
| HZ-01 | 機械的 | モータ過速度による脱水槽破損・飛散 | 傷害、打撲、骨折 |
| HZ-02 | 熱的 | モータロック時の過電流による発火・焼損 | 火傷、火災 |
| HZ-03 | 機械的 | 蓋開放中のモータ動作による巻き込み | 巻き込み傷害 |
| HZ-04 | 電気的 | 絶縁破損による漏電 | 感電 |
| HZ-05 | 電気的 | 電源断時の不正再起動 | 予期しないモータ動作 |
| HZ-06 | 制御的 | センサ故障による異常制御 | 過速度、停止不可 |

**注**: HZ-ID は PH-03（HARA）で正式にリスク評価し、SG-xx（安全目標）に紐付ける。

### 5.2 適用される安全関連法規・規格

| 規格・法規 | 該当項目 | 備考 |
|---------|-------|------|
| **IEC 60730-1:2023** | 自動電気制御装置の一般要件 | Class B 自己診断要件 |
| **IEC 60730-2-7** | 洗濯機用制御装置 | Class B 対象 |
| **IEC 60335-2-7:2019** | 洗濯機の安全 | 電気安全、EMC |
| **電気用品安全法（PSE）** | 家庭用電気機器の安全基準 | 日本国内向け |
| **ISO 26262:2018** | 自動車機能安全（プロセス基準として適用） | V-model + SPICE 基盤 |

### 5.3 サイバーセキュリティ考慮（段階的導入）

本プロジェクトの洗濯機がネットワーク接続機能（WiFi/BT）を持つ場合、PH-03 で以下の脅威分析を実施する:

| TH-ID | 脅威カテゴリ | 考慮事項 |
|-------|-----------|---------|
| TH-01 | 不正操作 | ネットワーク経由のモータ不正制御 |
| TH-02 | ファームウェア改竄 | OTA更新機構の悪用 |
| TH-03 | データ漏洩 | 使用パターン等の個人情報流出 |

**注**: 適用法規は `project.json` standard.security.applicable が確定次第、正式に反映する。
