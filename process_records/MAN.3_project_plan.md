## 5. スコープ定義

| 項目 | 内容 |
|------|------|
| プロジェクト名 | TORASAN 洗濯機BLDCモータ制御システム (WMC) |
| 目的 | IEC 60730 Class B 準拠の安全な洗濯機モータ制御ソフトウェアの開発 |
| 開発プロセス | ISO 26262 V-model + Automotive SPICE v3.1 |
| MCU | Renesas RL78/G14 (R5F104BG) — Flash 64KB / RAM 5.5KB / 32MHz |
| モータ | BLDC 1500W / 定格1200rpm / 定格6A |
| 安全レベル | ASIL B (プロセス) / IEC 60730 Class B (製品) |
| 開始日 | 2026-02-27 |
| 担当 | TORA（責任者）+ Claude Code（AI開発支援 TCL3） |

### スコープ内

| 成果物 | 説明 |
|--------|------|
| 安全分析 | HARA、FMEA、FSC、TSC |
| SW設計・実装 | SRS、アーキテクチャ、詳細設計、Cソースコード |
| 安全診断 | IEC 60730 Annex H 準拠の自動診断機能 |
| テスト | ユニット、統合、システム、安全検証テスト |
| 安全ケース | GSN ベースの Safety Case |

### スコープ外

- ハードウェア製造・量産
- 認証機関への正式申請
- 洗濯機本体（筐体・給排水系）の設計
