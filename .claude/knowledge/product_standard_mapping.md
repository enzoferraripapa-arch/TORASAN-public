## 2. 安全分類方法の詳細

### 2.1 ASIL判定 — 車載専用

- **適用規格**: ISO 26262 Part 3 Table 4
- **分類要素**: Severity(S) × Exposure(E) × Controllability(C) → ASIL QM/A/B/C/D
- **適用範囲**: 車載電子機器のみ。他製品カテゴリには適用不可
- **根拠**: 判定基準が道路交通の運用状況（走行速度・交通量等）に依存

### 2.2 IEC 60730 Class分類 — 家電制御機器

- **適用規格**: IEC 60730-1 §H.2
- **分類方法**: 制御機能の種別による分類（リスク評価ではない）
- **分類定義**:

| Class | 定義 | 機能例 |
|-------|------|--------|
| A | 安全に関係しない一般制御 | タイマー制御、表示制御 |
| B | 安全でない動作を防止する制御 | 温度制御、回転数制御、過電流保護 |
| C | 特殊危険を防止する制御 | ガス弁制御、高圧蒸気制御 |

- **決定手順**: 製品の制御機能を列挙 → 各機能の安全寄与度で分類
- **典型的な適用**: 洗濯機モータ制御 = Class B（安全でない高速回転を防止）

### 2.3 SIL判定 — 産業機器

- **適用規格**: IEC 61508 Part 5 (リスクグラフ) または Part 5 Annex D (リスクマトリクス)
- **分類方法**: リスクグラフ（C:結果, F:暴露頻度, P:回避可能性, W:需要率）
- **分類結果**: SIL 1/2/3/4（数字が大きいほど高要求）
- **適用範囲**: E/E/PE安全関連系が対象

### 2.4 IEC 62304 SWクラス — 医療機器

- **適用規格**: IEC 62304 §4.3
- **分類方法**: ソフトウェアの安全寄与度による
- **分類定義**: Class A=負傷なし, B=重傷でない負傷, C=死亡/重傷の可能性

## 4. 誤適用アンチパターン

| # | 誤り | 正しい対応 | 検出方法 |
|---|------|----------|---------|
| 1 | 非車載製品に ASIL 判定を適用 | 製品規格の分類方法を使用 | product_override != null → ASIL判定は非適用 |
| 2 | IEC 60730 製品に SIL 判定を適用 | Class A/B/C 分類を使用 | product_override = IEC60730 → SIL非適用 |
| 3 | Class B 製品に ASIL D 要件を適用 | Class B の要件 (Annex H) を適用 | product_override != null → ASIL D要件不要 |
| 4 | 成果物中に製品規格と不一致の分類用語 | 製品規格の分類用語に統一 | docs/ 内で ASIL と grep → product_override時は不整合 |
| 5 | プロセス手法の流用と製品分類の適用を混同 | 手法流用は OK、分類は製品規格に従う | HARA/FSC で ASIL 列があれば確認 |

## 5. プロセス手法の流用可否

ISO 26262 の V-model / SPICE / FMEA / FTTI 分析は、製品規格を問わず流用可能。
ただし安全分類（ASIL判定）は車載専用であり、他製品では以下の使い分けが必要:

| 項目 | 流用可 | 流用不可 |
|------|--------|---------|
| V-model開発プロセス | Yes | — |
| Automotive SPICE プロセス評価 | Yes | — |
| FMEA/FTA/FMEDA | Yes | — |
| FTTI分析 | Yes | — |
| S×E×C リスク評価（補足分析として） | Yes | — |
| ASIL判定（S×E×C→ASIL A/B/C/D） | — | 車載以外は不可 |
| Hardware Metrics (SPFM/LFM/PMHF) | Yes（目標値は規格による） | — |
| DC カテゴリ (Low/Medium/High) | Yes | — |

## 6. project.json 設定パターン

| カテゴリ | standard.asil | standard.product_override | standard.product_override_class |
|---------|--------------|--------------------------|-------------------------------|
| 車載 | D | null | null |
| 家電 | B | IEC60730 | ClassB |
| 産業 | C | IEC61508 | SIL2 |
| 医療 | — | IEC62304 | ClassC |
| 鉄道 | — | EN50128 | SIL4 |

**注意**: `standard.asil` はプロセス厳密度の指標として使用。product_override が設定されている場合、ASIL判定は製品分類としては適用されない。
